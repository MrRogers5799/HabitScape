## 🗂️ What Each Directory Contains

### `/src/screens`
User-facing screen components. Each screen is a top-level page:
- Auth flow (login/signup)
- Main app screens (Activities, Skills, Settings, Profile)
- Pre-built UI with all styling and business logic

### `/src/services`
Backend integration layer:
- Firebase configuration and initialization
- Authentication operations (signup, login, logout, user profile)
- Firestore CRUD operations (create, read, update)
- Real-time listeners for data synchronization

### `/src/context`
React Context providers for global state:
- AuthContext: Current user, auth state
- SkillsContext: All skills data, real-time updates
- ActivitiesContext: Activities list, completions, real-time updates

### `/src/components`
Reusable UI components:
- ProgressBar: Visual progress indicator

### `/src/types`
TypeScript type definitions:
- All interfaces used across the app
- Ensures type safety throughout

### `/src/utils`
Pure utility functions:
- XP calculations (level, progress, formatting)
- No side effects
- Easy to test and reuse

### `/src/constants`
App configuration and data:
- OSRS skills (names, icons, descriptions)
- XP progression table
- Pre-defined activities
- Cadence options and multipliers

### `/src/navigation`
Navigation structure:
- Root navigator (handles auth vs app navigation)
- Tab navigator (bottom tabs)
- Stack configuration
